# zhc-sporttery-resource

本仓库用于归档和发布多个独立的 HTML 页面。  
页面内容涵盖体育相关资源展示，不针对特定域名或网站。

## 项目简介

这是一个资源归档仓库，主要存放若干静态 HTML 文件。  
每个页面均为独立完整，不依赖外部数据库或后端服务。  
适用于展示、存档或离线预览场景。

## 目录说明

```
├── index.html          # 主入口页面（如有）
├── page1.html          # 示例页面 1
├── page2.html          # 示例页面 2
└── ...
```

页面数量与具体内容根据归档需要更新。  
所有文件均为标准 HTML 格式，可直接在浏览器中打开。

## 页面归档说明

- 每个 HTML 文件均为独立归档单元
- 文件命名简洁，不包含特殊字符
- 页面内资源（如样式、脚本）尽量内联，减少外部依赖
- 当前归档内容不涉及动态数据接口

## 维护说明

本仓库以归档为主要目的，更新频率不定。  
如需要提交新的页面或修正现有页面，欢迎提交 Pull Request。  
建议提交前确保 HTML 文件可独立运行。
